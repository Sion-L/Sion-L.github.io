Tomcat的搭建
2020-09-08 2 min read # tomcat的搭建

一、下载jdk的包，最好比tomcat高一个版本
1、把包放到/usr/local下
rpm -ivh +包名
java -version 检查版本

2、配置环境变量
vi /etc/profile
在下面添加：
JAVA_HOME=/usr/java/jdk-...
CLASSPATH=JAVAHOME/lib/PATH=JAVA_HOME/lib/ PATH=JAVAH​OME/lib/PATH=PATH:$JAVA_HOME/bin
export PATH JAVA_PATH CLASSPATH

3、验证安装结果
用文本编辑创建Test.java
vi Test.java 并输入以下代码
public class Test {
public static void main(String args[]) {
System.out.println("A new jdk test !")
}
}
编译并运行，成功输出，配置完成
#javac Test.java
#java Test
A new jdk test ! 代表成功了

如果运行出现找不到或无法加载主类的错误
编辑/etc/profile
